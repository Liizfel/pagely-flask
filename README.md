## 💡 Visão Geral do Projeto

O **Pagely** é um espaço digital minimalista criado para transformar a leitura em uma experiência visual e organizada. Desenvolvido para leitores que buscam **foco** e **clareza**, ele permite gerenciar sua estante virtual com uma interface suave e moderna.

O objetivo é metrificar o hábito de leitura sem perder a leveza, oferecendo uma visão analítica sobre o tempo dedicado aos livros e a evolução de cada obra.

---

## 🎯 Funcionalidades Principais

Aqui estão as ferramentas essenciais para a sua gestão literária:

### 🧭 Dashboard Analítico
A tela principal oferece uma visão 360º dos seus hábitos:
* **Estatísticas Rápidas:** Livros Lendo, Lidos, Quero Ler e Abandonados.
* **Volume de Leitura:** Gráfico de páginas lidas por dia na semana.
* **Preferências:** Gráfico de barras indicando seus gêneros literários mais consumidos.

### 📝 Módulos de Gestão

| Módulo | Descrição | Status |
| :--- | :--- | :--- |
| **Estante Digital** | 📚 Cadastro de livros com controle de **páginas totais e lidas**. | Implementado |
| **Gêneros Literários** | 🏷️ Categorização em Romance, Conto, Crônica, Drama e mais. | Implementado |
| **Top Ranking** | 🏆 Lista dos livros favoritos baseada na sua avaliação. | Implementado |
| **Diário de Resenhas** | ✍️ Espaço para notas críticas e sistema de avaliação por estrelas. | Implementado |

## ⚙️ Tecnologias Utilizadas

Ferramentas que tornam o Pagely possível:

* **React.js (Vite):** Estrutura de componentes de alta performance.
* **Tailwind CSS:** Estilização clean com paleta de cores equilibrada.
* **Recharts:** Visualização de dados de forma intuitiva.
* **Framer Motion:** Animações e micro-interações para uma UX fluida.
* **Lucide React:** Iconografia moderna e consistente.

## 🚀 Como Rodar o Projeto

Siga os passos para ter o Pagely rodando em sua máquina:

### 🛠 Instalação e Configuração

1. **Clone o Repositório:**
   ```bash
   git clone [https://github.com/Liizfel/pagely-flask.git](https://github.com/Liizfel/pagely-flask.git)
   cd pagely-flask
   ```

2. **Instale as Dependências:**
   ```bash
   npm install
   ```

3. **Inicie o Servidor de Desenvolvimento:**
   ```bash
   npm run dev
   ```
